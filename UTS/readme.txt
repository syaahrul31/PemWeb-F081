------------------------------  App Sen  ----------------------------------
AppSen merupakan aplikasi presensi berbasis web yang ditujukan bagi para karyawan di suatu perusahaan. 

Fitur App Sen ----------------------------------------------------------
App Sen menyediakan fitur check in bagi para karyawan yang ingin melakukan check in kehadiran sebelum bekerja. App Sen menyediakan layanan bagi admin untuk bisa mengolah data kehadiran karyawan. Data kehadiran karyawan dapat direpresentasikan dalam bentuk tabel dan diagram. Data pada tabel dapat diunduh dalam bentuk excel maupun PDF.

Cara Penggunaan ---------------------------------------------------

Oleh Karyawan
- Untuk melakukan presensi, langkah awal yang harus dilakukan karyawan adalah mengunjungi website App Sen pada tautan berikut "https://syaahrulhidayat.000webhostapp.com/finalproject/home.html"
- Di halaman beranda, tersedia 2 menu utama, yakni 'Admin' dan 'Karyawan'.
- Untuk melakukan check in, karyawan cukup mengklik menu 'Karyawan'
- Karyawan akan diarahkan pada halaman check in.
- Karyawan cukup mengisi dengan sesuai, form yang tersedia.
- Klik submit, dan kehadiran karyawan akan direkam untuk hari tersebut.

Oleh Admin
- Untuk mengakses web khusus admin, langkah awal yang harus dilakukan Admin adalah mengunjungi website App Sen pada tautan berikut "https://syaahrulhidayat.000webhostapp.com/finalproject/home.html"
- Untuk melakukan konfigurasi data kehadiran karyawan, Admin cukup mengklik menu 'Admin' kemudian melakukan login pengguna.
- Di beranda Admin, admin bisa melakukan konfigurasi data kehadiran karyawan dengan cara masuk ke menu 'Akun Pengguna'.
- Admin dapat melakukan konfigurasi data karyawan meliputi edit data dan hapus data.
-Admin dapat mendownload data yang tersedia dalam tabel dengan cara mengklik tombol 'excel' - jika data ingin di download dalam bentuk excel - atau tombol 'pdf' - jika data ingin di download dalam bentuk pdf -.

Login & Register ------------------------------------------------------------
- Untuk bisa memasuki halaman admin, diwajibkan melakukan login pengguna. Silahkan input username dan password sesuai dengan akun admin yang telah terdaftar.
- Jika belum memiliki akun, Admin bisa melakukan registrasi dengan mengklik tombol 'sign up.'
- Untuk mendaftarkan diri, silahkan mengisi form yang tersedia di halaman sign up, kemudian klik 'submit.'
- Akun Admin telah terdaftar
